◆ uhos_wifi_disconnect()

uhos_s32 uhos_wifi_disconnect ( void  )
extern

断开WIFI连接。

如果是STA模式，取消当前正在进行的连接或者断开已经存在的连接；如果是AP模式，断开当前所有连接。断开WIFI连接后，不能自动再进行新的连接。

返回
成功返回0，失败返回-1。
